PATCH CONTENTS

This patch adds:
- Dice roller panel in the lower-left sidebar
- Click-to-increment dice selection (D4, D6, D8, D10, D12, D20)
- Roll button
- Clear button
- Cleaner multi-line roll log
- Narrate ON/OFF toggle
- Backend support for narrateMode so AI can resolve scenes more softly and narratively when enabled

FILES INCLUDED
- server.js
- web/index.html
- web/style.css
- web/app.js

INSTALL
Copy these files over the existing project files, replacing the old ones.
Then run start.bat again.
